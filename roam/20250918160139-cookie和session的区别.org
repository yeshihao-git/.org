#+title: Token、Cookie、Session
#+filetags: network

* Token、Cookie、Session
- Token :: 服务端发给客户端的 *身份凭证* ，用于 *身份验证*
  1. 存在 客户端

- Cookie :: 存储 *用户信息* 的文本文件，发送 HTTP请求 时带上Cookie，实现 *用户信息、用户状态管理*
  1. 存在 客户端浏览器

- Session :: 服务端为每个客户端创建的专属会话，存储用户的会话信息（eg：登录信息、权限数据）
  1. 存在 服务端
