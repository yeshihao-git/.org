* TCP/IP网络模型
1. 应用层    ：给用户提供服务（eg：网页浏览、文件传输）， =报文（message）=
2. 传输层    ：端到端数据传输，可靠通信（[[id:2734ca58-d8c8-48e3-a490-1aed5467b9b5][TCP]]）或高效通信（UDP）， =报文段（segment） = 报文 + tcp头=
3. 网络层    ：跨网络通信的主机间通信， =包（packet） = 报文段 + ip头=
4. 网络接口层：物理网络中传输数据， =帧（frame） = 包 + 帧头 + 帧尾=
   - 帧头 :: 数据链路层的头部信息，包含：[[id:1ed3f34a-88a1-419a-8879-cb9141dc461c][MAC]]头(以太网帧头)、PPP帧头、802.11帧头(Wi-Fi)
