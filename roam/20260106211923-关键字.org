* 关键字
- 关键字 :: C++中保留的标识符，有特殊含义，不能被用作标识符


** 说明符
:PROPERTIES:
:ID:       200675d1-7b3f-421a-bd50-606703b37fd0
:END:
- 说明符 :: 关键字的子集，用于说明变量/函数特性。分为：
  1. 存储类说明符
  2. 函数说明符
  3. 访问说明符
  4. 其他说明符

- 存储类说明符 :: 说明 对象的[[id:853a3bee-b823-49fc-acd6-804eecb74822][存储期、链接性]]
|--------------+---------------------------------------+---------------------------|
| 说明符       | 含义                                  | 注释                      |
|--------------+---------------------------------------+---------------------------|
| [[id:c2be420e-464a-4713-8ebc-19e943a0cf67][extern]]       | 静态存储期，外部链接性                |                           |
| [[id:be24bcfa-3a9e-4e52-93e4-66163f87f280][static]]       | 静态存储期，内部链接性                |                           |
| thread_local | 变量存储期和线程绑定                  |                           |
| mutable      | 即使是 const，也允许修改              |                           |
| [[id:bee683d2-eec4-45a8-af02-be369d182e8d][auto]]         | 自动存储期                            | C++11后不属于存储类说明符 |
| register     | 自动存储期 + 提示编译器将其放入寄存器 | C++17后不属于存储类说明符 |
|--------------+---------------------------------------+---------------------------|

- 函数说明符 :: 说明 函数特性
|----------+--------------------------|
| 说明符   | 含义                     |
|----------+--------------------------|
| [[id:3e85303c-39c7-4ac3-9d88-a9513d036477][inline]]   | 内联                     |
| virtual  | [[id:b23cd926-82ec-489e-ad8f-96b86e5559c2][虚函数]]、[[id:09c9cdf8-61a3-48c9-86e3-4caa29fbdc05][纯虚函数]]、[[id:9f071537-cd60-49c4-97bc-47e3474b7c2e][虚基类]] |
| [[id:2f9e4a5c-e6fe-4006-a6df-0bbd7fce0ded][explicit]] | 禁止隐式转换             |
|----------+--------------------------|

- 访问说明符 :: 说明 类成员访问权限
|-----------+------|
| 说明符    | 含义 |
|-----------+------|
| public    | 公有 |
| private   | 私有 |
| protected | 保护 |
|-----------+------|


*** 类型限定符
:PROPERTIES:
:ID:       91e55c3a-c98d-4907-874c-453f5444b82a
:END:
- 类型限定符 :: 说明符的子集，用于限制 *访问/修改* 规则
  1. 🔥 *是类型的一部分* （eg：int 和 const int 是不同类型）
|----------+----------------------|
| 说明符   | 含义                 |
|----------+----------------------|
| [[id:1450514f-880d-42f5-8bb4-f114c1a5b675][const]]    | 常量，值不可修改     |
| volatile | 易变，防止编译器优化 |
|----------+----------------------|
| restrict | C语言的类型限定符    |
|----------+----------------------|
