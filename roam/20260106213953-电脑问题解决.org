#+title: 电脑问题解决

* VIA改键
:PROPERTIES:
:ID:       7a80e9a7-c1f7-4964-9333-85e042fc9cae
:END:
1. 将键盘用 USB 插入电脑， =Fn + t= 改为有线模式
2. 进入 VIA网站（注意： *不是官方网站* ） 改键即可：https://via.toqmk.com/ （VIA 对键盘按键的修改是直接 *写入键盘固件* 里的）


* 华硕B850-G找不到蓝牙设备
:PROPERTIES:
:ID:       488031cd-016a-4c5e-89ea-495ac06abe18
:END:
1. [[https://rog.asus.com.cn/motherboards/rog-strix/rog-strix-b850-g-gaming-wifi-s/helpdesk_download/][官网]] 下载驱动
2. 解压缩
3. 双击 =AsusSetup.exe=


* 网卡rtw89_8922ae PHY c2h class 2 not support问题
:PROPERTIES:
:ID:       90fedf95-2548-4b6c-983a-f24949ff907b
:END:
#+begin_src bash
# sudo dmesg 可以发现这个问题，原因在于驱动版本老旧
git clone https://aur.archlinux.org/rtw89-dkms-git.git
cd rtw89-dkms-git
makepkg -sri
reboot
#+end_src


* windows关闭快速启动和休眠
:PROPERTIES:
:ID:       a5065cca-741b-4588-acb2-e443102c5fb1
:END:
1. 管理员权限运行 PowerShell
2. =powercfg /H off= 关闭
3. =powercfg /a= 验证
