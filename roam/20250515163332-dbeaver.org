** DBeaver 连接 MySQL 时 "Public Key Retrieval is not allowed" 错误
:PROPERTIES:
:ID:       ab5dc9de-96a6-4d94-9444-c1a7899d3e86
:END:
- 原因 :: mysql服务器可能启用了SSL或RSA加密，但DBeaver默认不允许检索公钥
- 解决 ::
  1. 编辑连接 => 驱动属性 => allowPublicKeyRetrieval选项，设置为true
