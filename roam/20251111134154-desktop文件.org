#+title: .desktop文件
#+filetags: linux

* .desktop文件
1. 用于让 Linux桌面环境（KDE、GNOME等）和 相关工具（Krunner、应用菜单等） *识别、索引、启动* 目标程序（AppImage）
2. .desktop文件的格式和作用，见 https://www.freedesktop.org/wiki/

** 快速开始
1. 进入 =~/.local/share/applications=

2. 编写.desktop文件
#+begin_example bash
[Desktop Entry]
Name=应用名称  # 例如：MyApp
Comment=应用描述  # 例如：A powerful tool for...
Exec=/path/to/your/appimage.AppImage  # AppImage 文件的绝对路径（必填）
Icon=/path/to/icon.png  # 应用图标路径（可选，支持 png/svg）
Terminal=false  # 是否在终端中运行（通常为 false）
Type=Application
Categories=Utility;Development;  # 应用分类（影响菜单归类，可选）
Keywords=关键词1;关键词2;  # 用于 Krunner 搜索的关键词（可选）
#+end_example

3. 刷新应用索引
#+begin_example bash
update-desktop-database ~/.local/share/applications/
#+end_example
