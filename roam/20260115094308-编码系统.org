* ASCII
:PROPERTIES:
:ID:       0a7a1469-d7f7-4720-9fbd-00f2e621c624
:END:
- ASCII :: （美国信息交换标准代码）基础的单字节字符编码方法
  1. 1字节里的7位（0-127：能表示128个字符）将整数 -映射-> 字符，在 C++ 中用 [[id:aa05eb91-78df-463b-a6a4-9111833391e8][char]]类型 表示
  2. 能覆盖 英文字母、数字、常见标点、一些控制字符
  3. 不能满足多语言的编码需求：中文、日文
#+begin_example bash
'A'-'Z'       # 65 - 90
'a'-'z'       # 97 - 122
#+end_example


* UTF-8
:PROPERTIES:
:ID:       31d36e0d-adb4-451a-ad93-a00956f7a4d7
:END:
- UTF-8 :: Unicode是字符集，为全球所有字符（eg：中文、日文、英文等）分配一个唯一的数字编号（称为码点），但没规定码点怎么存储在计算机里。UTF-8就是一套存储规则，定义如何将 Unicode的码点转换为 1-4字节的二进制数据存储（或者说如何用 1-4字节 表示一个字符）
  1. 兼容 [[id:0a7a1469-d7f7-4720-9fbd-00f2e621c624][ASCII]]
  2. UTF-16 2或4字节编码，UTF-32 固定4字节编码
  3. cpp 中用 =char8_t= 表示 UTF-8， =char16_t= 表示 UTF-16， =char32_t= 表示 UTF-32
